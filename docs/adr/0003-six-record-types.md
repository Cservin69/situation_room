# ADR 0003 — Six record types

**Status**: Accepted
**Date**: 2026-04-19

## Decision

Stockpile recognizes exactly six fundamental record shapes: Observation,
Event, Entity, Relation, Document, Assertion. Geometry is a field on
Entity/Event/Observation, not a top-level type.
